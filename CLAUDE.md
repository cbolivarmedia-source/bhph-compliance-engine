# BHPH Compliance Engine

## Git Workflow

- After completing each phase of work, merge the worktree/feature branch into main and push to origin. Do not leave completed work in isolated worktree branches.
- Commit and push after every passing task, not just at the end of a phase.
- Before reporting a phase as complete, confirm the work is on main and pushed to GitHub.
- If using git worktrees, run `git push origin <branch>` before removing the worktree so the branch is recoverable from remote even if the local worktree is cleaned up.
